## Request
HR requested a Microsoft 365 account for a new employee joining the Marketing department:
- **Name:** Jonas Müller
- **Department:** Marketing
- **Manager:** s.klein@corp.local
- **Start date:** Monday 07.07.2026

## Actions Taken
- Logged into **Microsoft 365 Admin Center** (admin.microsoft.com)
- Navigated to **Users → Active users → Add a user**
- Created the user account:
  - Display name: Jonas Müller
  - Username: j.mueller@corp.local
  - Set temporary password and enabled "User must change password at first sign-in"
- Assigned **Microsoft 365 Business Standard** licence
- Added user to the following Microsoft 365 groups:
  - Marketing department group
  - Company-wide Teams channel
- Configured **Outlook mailbox** — confirmed mailbox provisioned successfully
- Added user to relevant **SharePoint** site for Marketing documents
- Sent welcome email to HR with login credentials and M365 onboarding guide

## Result
Microsoft 365 account fully configured and ready before the employee start date. HR confirmed receipt of credentials. User able to sign in and access Outlook, Teams, and SharePoint on first login.
